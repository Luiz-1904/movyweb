# Movy Landing Page

Landing page institucional da Movy, criada com **Node.js**, **Vite**, **TypeScript** e **Tailwind CSS**.

## Requisitos

- Node.js 18+
- npm

## Como rodar localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:5173` no navegador.

## Build de produção

```bash
npm run build
npm run preview
```

## Estrutura

- `index.html`: markup principal.
- `src/main.ts`: inicialização do app (scripts simples).
- `src/style.css`: estilos com Tailwind.
- `tailwind.config.ts`: tema e utilitários personalizados.
